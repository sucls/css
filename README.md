## CSS

一个css相关的学习项目

less  scss sass

### 相关插件

+ Live Sass Compiler
+ Live Server

### 启动

  + 基于 Live Server
    ```
      右下角 'Go Live'
    ```
  + 基于Express
    ```bash
      node ./serve.js
    ```

### 应用

  + 布局 (layout)
    - 传统布局 (/layout/modern.html)
    - 图片瀑布流布局 (/layout/image-cassade.html)
